## Setup

1.  Clone the repo:
    ```bash
    git clone https://github.com/redwanshahriarshubho/homenest-client.git
    cd homenest-client
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.   Start the development server:
    ```bash
    npm run dev
    ```
